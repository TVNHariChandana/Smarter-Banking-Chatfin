💬 Smarter Banking Chatfin

📌 Description

This is a chatbot application designed to answer banking-related queries using a trained model and dataset.

🚀 Features

* Answers user queries related to banking
* Uses NLP-based chatbot logic
* Simple and interactive interface

🛠️ Technologies Used

* Python
* Django
* NumPy
* NLTK

⚙️ Setup Instructions

1. Clone the repository
2. Install required libraries
3. Run the server:
   python manage.py runserver

⚠️ Important Note

The file `answer.npy` is not included due to size limitations.

👉 Download it from here:
https://drive.google.com/file/d/1YfecUZP1k5yAzPzdz02zicgIbIh4AN5w/view?usp=drive_link

After downloading, place it inside the `model/` folder.

📂 Project Structure

* chatbot/ → main project
* chatbotapp/ → application logic
* dataset/ → training data
* model/ → trained model files
